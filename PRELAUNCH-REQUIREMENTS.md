# Pre-Launch Requirements

## Apple Developer Program License
- [ ] Enroll in the Apple Developer Program at developer.apple.com/programs/enroll ($99/year)
- [ ] Required to unlock iCloud + CloudKit capabilities in Xcode
- [ ] Required to publish the app on the App Store

## iCloud / CloudKit Setup (do after enrolling)
- [ ] Open Xcode → select the Note-taking target → Signing & Capabilities tab
- [ ] Click + Capability → search "iCloud" → add it
- [ ] Check the CloudKit checkbox
- [ ] Commit the auto-generated Note-taking.entitlements file to the repo
